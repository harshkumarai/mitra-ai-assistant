# 🤖 Mitra AI Assistant

Mitra is an AI-powered virtual assistant built using Python.

## Features

- Voice Assistant
- AI Chat Support
- Memory System
- Command Execution
- Automation Tasks
- Modern Modular Architecture

## Tech Stack

- Python
- AI APIs
- Speech Recognition
- Automation Tools

## Project Structure

```
commands/
memory/
utils/
assets/
```

## Installation

```bash
git clone https://github.com/harshkumarai/mitra-ai-assistant.git
cd mitra-ai-assistant
pip install -r requirements.txt
```
